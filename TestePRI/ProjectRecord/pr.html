Resume

  • Exercício Nº 1: Prémios Nobel

  • - Neste exercicio é nos fornecido um dataset em formato JSON, o mesmo encontra-se com erros na sua formatação, tendo os corrigido no Visual Studio Code um a um, ou seja, manualmente, estes erros consistiem em ","(virgulas) mal colocadas, bem como tabs mal definidos e enters mal estruturados, tendo corrigido esses erros de sintaxe do JSON. Para a persintência dos dados fiz uma unica modificação no dataset, substituindo o atributo id por _id

  • - Utilizei para este exercicio para guardar o dataset o mongo, utilizando o seguinte comando: mongoimport --db jcrpubs --collection jcrpubs --file jcrpubs.json --jsonArray Como expliquei em cima ao fazer aquela substituição fiz com o data set ao ser importado para o mongo tenha persistência. Tendo obtido no final o total de 120 registos guardados.

  • - Neste caso utilzei o mongo pelas querys que irão ser feitas ao dataset, em que o mongo nos permite com facilidade a sua resolução.

  • - O servidor é capaz de responder as querys todas pedidas no enunciado

  • Exercício Nº 2: CLAV: Classificação e Avaliação da Informação Pública

  • - Neste exercicio é nos pedido para utilizar uma API de dados online e a respetiva construção de uma interface para o mesmo, para a resolução deste exercicio optei por utilizar o PUG em junção com o W3

  • - As respostas paras as respectivas perguntas a alínea 1, irão dentro do projeto e será em baixo mostrado o link para aceder as mesmas.

  • - É uma interface simples, utilizando um indici inicial se pretende pesquisar por entidades ou tipologias, de seguida temos uma lista com a respetiva informção que foi escolhia anteriormente, e de seguida temos uma página detalhada se cada informação respetiva, sendo possivel a visualização de entidades para tipologias e vice-versa.
