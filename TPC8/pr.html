Resume

  • Neste trabalho prático foi pedido para desenvoler um servidor baseado em Node.js e usando o Express sobre Gestão de Ficheiros, o mesmo terá que guardar ficheiros e consulta-los.

  • O servidor tem de ser capaz de adicionar múltiplos ficheiros de uma so vez, e efectuar a consulta do mesmo através dum modal.

  • O mesmo usa como front-end a interface pug que nos permite uma fácil construção. Também usa Express para ser mais fácil a criação do mesmo servidor. O mesmo servidor enconta-se operacional e utiliza a porta 5020.
