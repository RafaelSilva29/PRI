Resume

  • Neste trabalho prático foi pedido para desenvoler um servidor baseado em Node.js e usando o Express sobre Gestão de Alumos.

  • O servidor tem de ser capaz de remover, visualizar e adicionar alunos, bem como remover e adicionar as notas ao respetivo aluno.

  • O mesmo usa como front-end a interface pug que nos permite uma fácil construção. Também usa Express para ser mais fácil a criação do mesmo servidor. O mesmo servidor enconta-se operacional e utiliza a porta 3029.
