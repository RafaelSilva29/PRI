Resume

  • Neste trabalho prático foi pedido para desenvoler um servidor baseado em Node.js para o tratamento do DataSet do Arquivo de Musicas Evo.

  • De seguida foi pedido pedido para desenvolver dois StyleSheets, o primeiro para separar o DataSet em XML individuais e o segundo foi para converter XML para HTML, para o servidor poder fazer a conversão individual de cada uma dos ficheiros.

  • Sendo assim o servidor é capaz de responder aos pedidos que lhe chegam, bem como no tratamento deles.

  • O servidor enconta-se em localhost:1234 e o pedido é feito em localhost:1234/musica/{id} sendo que o id é o número da música.
